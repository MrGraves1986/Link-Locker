About Link Locker

Link locker was created with the goal of helping a user stay organized in a world where there is quite literally an infinite amount of data being produced daily. Instead of cluttering your desktop notes or bookmarks, you can store all those random links in one place and read them as time permits. Updates to this version will include automated emails via Celery which can be used in Django to automate/schedule happenings.
